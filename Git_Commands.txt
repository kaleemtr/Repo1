Git Commands

git --version  (to check git version)
git init   (to add folder to git)
git status  (to check files and folder status)
dir > test1.txt  (to add file to the project1 folder)
git add test1.txt  (to add file to git start tracking)
git commit -m "...."  (to commit changes with the message (-m))
git remote add origin https://github.com/kaleemtr/Repo1.git  (to add GitHub account to git)
git config --global user.email "kaleemtr@gmail.com"  (to add GitHub user email to git)
git config --global user.name "kaleemtr"  (to add GitHub user name to git)
